# Tax Preparation Software Platform

This repository contains the initial scaffolding for a full-stack tax preparation platform with a FastAPI backend and a React (Vite + TypeScript) frontend.

## Structure

```
├── backend/

│   │   ├── main.py

│   │   └── services/
│   ├── tests/
│   ├── requirements.txt
│   └── README.md
├── frontend/
│   ├── src/
│   ├── public/
│   ├── package.json
│   └── README.md
├── .github/
│   └── workflows/
│       └── ci.yml
└── README.md
```
